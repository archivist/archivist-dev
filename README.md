# Archivist Dev

## Install

Clone the repository.

```bash
$ git clone https://github.com/substance/archivist-dev.git
```

Navigate to the source directory.

```bash
$ cd archivist-dev
```

Install via npm

```bash
$ npm install
```

Start the dev server

```bash
$ npm run start
```

And navigate to [http://localhost:5000](http://localhost:5000)
